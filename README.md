# A-SOUL 五人混搭导航语音包混搭 Alpha 版本

创意工坊地址：[https://steamcommunity.com/...](https://steamcommunity.com/sharedfiles/filedetails/?id=2740270337)

本仓库存放工程文件（不含语音源文件，因为涉及到版权问题），同时 issues 区开放给大家提问题和建议
